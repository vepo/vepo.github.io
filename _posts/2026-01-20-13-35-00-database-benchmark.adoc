O Oracle é a base de dados comercial mais antiga que existe. Surgiu ainda em 1978 e vem sendo desenvolvida até hoje. Escrita em Assembly, C, C++ e Java.

== 2. MySQL

O MySQL é uma base de dados muito famosa que iniciou como um projeto Open Source desenvolvida pela empresa MySQL AB e depois foi adquirida pela Sun Microsystems que depois foi integrada a Oracle.
