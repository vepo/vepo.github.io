Alguns livros que recomendo. Não só sobre Desenvolvimento de Software.

Java efetivo: as melhores práticas para a plataforma

Joshua Bloch

Todo desenvolvedor Java deveria ler esse livro. Ele vai ensinar quais são as melhores práticas para desenvolvimento Java a partir do ponto de vista de quem conhece a linguagem internamente.

Como ser um Programador Melhor: um Manual Para Programadores que se Importam com Código

Pete Goodliffe

Esse livro aborda todas as áreas do desenvolvimento de software, ensinando boas práticas. Desde do desenvolvimento, preocupação com o código até a troubleshooting. Altamente recomendável!

Software Architecture in Practice

Len Bass, Paul Clements e Rick Kazman

Esse livro apresenta o que é Arquitetura de Software. Altamente recomendável para quem quer ser arquiteto.

A Philosophy of Software Design

John K. Ousterhout

Esse livro leva a discussão sobre design de código a um outro nível. Irá refletir sobre o que é complexidade e como ela pode influenciar no nosso código. Recomendo!

The Technological Society

Jacques Ellul

Livro questiona o uso da Técnica na sociedade moderna. Define o que é Técnica e sua relação com a Tecnologia. Classifica e analisa Técnicas largamente utilizadas na sociedade.
A Técnica transforma tudo o que toca em Máquina.
